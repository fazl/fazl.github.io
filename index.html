Adventures in CSS/HTML/JavaScript

I'm a serverside guy normally.

To get my teeth into HTML/CSS/JavaScript (..) I thought I'd build a tictactoe game in a single HTML file.

Here it is with variants, all of which are just a single static HTML file. No fancy Node.JS server or massive 25MB of angular libs needed!

They are actually about 6KB in size..

  • the basic 3 x 3 Tic Tac Toe, and
  • the resizable n x n Tic Tac Toe, and
  • the Sliding Tile Puzzle
Enjoy!
