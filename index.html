Adventures in CSS/HTML/JavaScript

I'm a serverside guy normally.

To get my teeth into HTML/CSS/JavaScript (..) I thought I'd build a tictactoe game in a single HTML file, after I saw an angular tutorial that built a simple tictactoe that took up about 25MB of disk space...

Each of these variants are a single static HTML file, merely a few KB in size.

  • the basic 3 x 3 Tic Tac Toe, and
  • the resizable n x n Tic Tac Toe, and
  • the Tile Puzzle
